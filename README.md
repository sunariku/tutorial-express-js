# Perkuliahan Backend Web Development

## Persiapan

- Node JS LTS
- MySQL DB Server Minimal Versi 8
- VS Code

## Penggunaan

- Clone repositori.
- Jalankan perintah `npm i` pada folder project.
- Import database `pos.sql`.
- Sesuaikan informasi credential database.
- Jalankan server `node index.js`.
